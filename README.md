# EatFrog
EatFrog is a library to build executables with automatic Assembler creation.

Current Work in Progress:

Platforms:
[-] Chip8
[-] x86

Plans:
- Retro-Consoles (nes, gb, gba)
- Managed Languages (.Net, Java)
- Webassembly
